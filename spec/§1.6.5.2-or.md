#### or

Returns `false` if both operands evaluate to `false`, `true` if either operand evaluates to `true`, and empty (`{ }`) otherwise:

|or |true |false |empty |
| - | - | - | - |
|**true** |`true` |`true` |`true` |
|**false** |`true` |`false` |empty (`{ }`) |
|**empty** |`true` |empty (`{ }`) |empty (`{ }`) |
