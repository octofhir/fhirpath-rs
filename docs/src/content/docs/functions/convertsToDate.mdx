# convertsToDate()

**Category**: Conversion

Returns true if the input can be converted to a date

## Signature

- Return type: `Boolean`

## Examples

```fhirpath
'2023-12-25'.convertsToDate()
```

```fhirpath
'invalid-date'.convertsToDate()
```
