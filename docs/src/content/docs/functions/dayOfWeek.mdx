# dayOfWeek()

**Category**: Date & Time

Returns the day of the week (0=Sunday, 1=Monday, ..., 6=Saturday)

## Signature

- Return type: `Integer`

## Examples

```fhirpath
Patient.birthDate.dayOfWeek()
```

```fhirpath
now().dayOfWeek()
```

```fhirpath
@2023-05-15.dayOfWeek()
```
