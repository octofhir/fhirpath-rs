# convertsToDecimal()

**Category**: Conversion

Returns true if the input can be converted to a decimal

## Signature

- Return type: `Boolean`

## Examples

```fhirpath
'123.45'.convertsToDecimal()
```

```fhirpath
'abc'.convertsToDecimal()
```

```fhirpath
123.convertsToDecimal()
```
