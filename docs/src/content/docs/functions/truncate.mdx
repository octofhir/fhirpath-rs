# truncate()

**Category**: Math

Returns the integer part of the input (truncation towards zero)

## Signature

- Return type: `integer`

## Examples

```fhirpath
3.14.truncate()
```

```fhirpath
(-2.9).truncate()
```

```fhirpath
5.truncate()
```
