# timezoneOffsetOf()

**Category**: Date & Time

Returns the timezone offset component of a DateTime in minutes

## Signature

- Return type: `Integer`

## Examples

```fhirpath
now().timezoneOffsetOf()
```

```fhirpath
@2023-05-15T14:30:45-05:00.timezoneOffsetOf()
```
