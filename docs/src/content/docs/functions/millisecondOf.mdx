# millisecondOf()

**Category**: Date & Time

Returns the millisecond component of a DateTime or Time

## Signature

- Return type: `Integer`

## Examples

```fhirpath
now().millisecondOf()
```

```fhirpath
@2023-05-15T14:30:45.123.millisecondOf()
```
