# convertsToQuantity()

**Category**: Conversion

Returns true if the input can be converted to a quantity

## Signature

- Return type: `Boolean`

## Examples

```fhirpath
'10 kg'.convertsToQuantity()
```

```fhirpath
'123'.convertsToQuantity()
```

```fhirpath
'abc'.convertsToQuantity()
```
