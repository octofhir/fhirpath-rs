# monthOf()

**Category**: Date & Time

Returns the month component of a Date or DateTime

## Signature

- Return type: `Integer`

## Examples

```fhirpath
Patient.birthDate.monthOf()
```

```fhirpath
now().monthOf()
```

```fhirpath
@2023-05-15.monthOf()
```
