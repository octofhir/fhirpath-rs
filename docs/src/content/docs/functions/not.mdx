# not()

**Category**: Logic

Returns true if the input is false, false if the input is true, and empty if the input is empty

## Signature

- Return type: `boolean`

## Examples

```fhirpath
true.not()
```

```fhirpath
false.not()
```

```fhirpath
Patient.active.not()
```
