# convertsToBoolean()

**Category**: Conversion

Returns true if the input can be converted to a boolean

## Signature

- Return type: `Boolean`

## Examples

```fhirpath
'true'.convertsToBoolean()
```

```fhirpath
'false'.convertsToBoolean()
```

```fhirpath
'invalid'.convertsToBoolean()
```
