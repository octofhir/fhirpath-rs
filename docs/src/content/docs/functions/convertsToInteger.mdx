# convertsToInteger()

**Category**: Conversion

Returns true if the input can be converted to an integer

## Signature

- Return type: `Boolean`

## Examples

```fhirpath
'123'.convertsToInteger()
```

```fhirpath
'abc'.convertsToInteger()
```

```fhirpath
true.convertsToInteger()
```
