# toQuantity()

**Category**: Conversion

Converts the input value to a quantity

## Signature

- Return type: `Quantity`

## Examples

```fhirpath
'10 kg'.toQuantity()
```

```fhirpath
42.toQuantity()
```

```fhirpath
'98.6 degF'.toQuantity()
```
