# yearOf()

**Category**: Date & Time

Returns the year component of a Date or DateTime

## Signature

- Return type: `Integer`

## Examples

```fhirpath
Patient.birthDate.yearOf()
```

```fhirpath
now().yearOf()
```

```fhirpath
@2023-05-15.yearOf()
```
