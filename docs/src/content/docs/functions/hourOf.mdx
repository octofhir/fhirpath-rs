# hourOf()

**Category**: Date & Time

Returns the hour component of a DateTime or Time

## Signature

- Return type: `Integer`

## Examples

```fhirpath
now().hourOf()
```

```fhirpath
@2023-05-15T14:30:45.hourOf()
```

```fhirpath
@T14:30:45.hourOf()
```
