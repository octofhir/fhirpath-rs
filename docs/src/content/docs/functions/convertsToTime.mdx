# convertsToTime()

**Category**: Conversion

Returns true if the input can be converted to a time

## Signature

- Return type: `Boolean`

## Examples

```fhirpath
'10:30:00'.convertsToTime()
```

```fhirpath
'invalid'.convertsToTime()
```
