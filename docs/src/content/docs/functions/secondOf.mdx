# secondOf()

**Category**: Date & Time

Returns the second component of a DateTime or Time

## Signature

- Return type: `Integer`

## Examples

```fhirpath
now().secondOf()
```

```fhirpath
@2023-05-15T14:30:45.secondOf()
```

```fhirpath
@T14:30:45.secondOf()
```
