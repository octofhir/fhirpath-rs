# dayOfYear()

**Category**: Date & Time

Returns the day of the year (1-366)

## Signature

- Return type: `Integer`

## Examples

```fhirpath
Patient.birthDate.dayOfYear()
```

```fhirpath
now().dayOfYear()
```

```fhirpath
@2023-05-15.dayOfYear()
```
