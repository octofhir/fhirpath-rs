# convertsToDateTime()

**Category**: Conversion

Returns true if the input can be converted to a datetime

## Signature

- Return type: `Boolean`

## Examples

```fhirpath
'2023-12-25T10:30:00'.convertsToDateTime()
```

```fhirpath
'invalid'.convertsToDateTime()
```
