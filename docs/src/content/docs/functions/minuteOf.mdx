# minuteOf()

**Category**: Date & Time

Returns the minute component of a DateTime or Time

## Signature

- Return type: `Integer`

## Examples

```fhirpath
now().minuteOf()
```

```fhirpath
@2023-05-15T14:30:45.minuteOf()
```

```fhirpath
@T14:30:45.minuteOf()
```
