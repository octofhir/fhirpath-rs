# FP0075 — Invalid datetime format

**Category**: Evaluation

## Description

The datetime string does not match the expected ISO 8601 format.

## Help

Use the format YYYY-MM-DDTHH:MM:SSZ or YYYY-MM-DDTHH:MM:SS+HH:MM for datetimes.
