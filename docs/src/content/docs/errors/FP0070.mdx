# FP0070 - Invalid date format

**Category**: Evaluation

## Description

The date string does not match the expected FHIR date format.

## Help

Use the format YYYY-MM-DD for dates or YYYY-MM-DDTHH:MM:SSZ for datetimes. Check for typos in the date string.
